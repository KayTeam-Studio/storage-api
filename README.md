# storage-api

## Maven
```XML
<repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
</repository>
```

```XML
<dependency>
    <groupId>com.github.KayTeam-Studio</groupId>
    <artifactId>storage-api</artifactId>
    <version>VERSION</version>
</dependency>
```

dependencies {
